# Lightning Fingers - Speed Type Test Application

A simple **Typing Test Application** built with Python 3.11 and Tkinter. This app helps users test their typing speed and accuracy by displaying sentences based on difficulty levels.

---

## Features

- **Select Difficulty Level**: Choose from Easy, Medium, or Hard difficulty levels.
- **Random Sentences**: Sentences are loaded dynamically based on the selected difficulty.
- **Countdown Timer**: Test your speed with a 60-second timer.
- **Real-Time Feedback**: Highlight correct (green) and incorrect (red) characters as you type.
- **Results Display**: At the end of the timer or when the sentence is completed, see matched and unmatched characters in a popup.

---

## Requirements

- Python 3.11 (no external dependencies required).
- The Tkinter library is included with Python.

---

## How to Run

1. Clone this repository or download the source code.
2. Ensure Python 3.11 is installed on your system.
3. Navigate to the project directory in your terminal.
4. Run the application using:
```bash
python main.py
```
